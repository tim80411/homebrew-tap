# Homebrew Tap

This is a [Homebrew](https://brew.sh/) tap for tim80411's tools.

## Installation

```bash
brew tap tim80411/tap
brew install ccx
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| `ccx` | CLI tool for managing multiple Claude Code settings profiles |

## Updating

```bash
brew update
brew upgrade ccx
```
